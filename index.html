MASS AI Project

AI-powered electricity theft detection for smart meter infrastructure.

MASS-AI is a production-ready machine learning platform that detects suspicious consumption behavior, visualizes anomalies across Turkey, compares normal vs theft-like time-series patterns, and presents analyst-ready model insights through a five-tab web dashboard.

Demo note: The current website showcases a bundled 2,000-customer synthetic snapshot. The underlying engine can generate new synthetic scenarios programmatically and can also be connected to external CSV / Excel data when needed.

Demo Data Note

This website currently runs on a packaged synthetic demo dataset.

The current web build ships with a bundled 2,000-customer synthetic dataset so the full dashboard can be reviewed immediately without using live customer records. MASS-AI also includes programmatic synthetic data generation logic for creating new scenarios, and the platform can be connected to external CSV / Excel datasets instead of the bundled demo files.

Current web demo Preloaded 2,000-customer synthetic feature snapshot for stable walkthroughs, presentations, and company-facing demos.

External data ingestion The same platform can be adapted to run on imported CSV / Excel data instead of the packaged synthetic sample files.

Key Features

Designed like a product, not just an experiment.

MASS-AI combines anomaly detection, explainability, customer review, and regional analytics into one web-first utility dashboard.

01

Multi-model detection

Random Forest, Isolation Forest, XGBoost, Gradient Boosting, and LSTM-based research paths support the broader detection story.

02

Regional monitoring

Suspicious customers are visualized geographically across Turkey with profile-aware filtering and theft thresholds.

03

Time-series comparison

Normal versus theft-like behavior is shown directly, which makes suspicious patterns easy to explain visually.

04

Customer drill-down

Inspect one account at a time with profile context, anomaly score, theft type, and behavioral evidence.

05

Interactive evaluation layer

ROC, PR, confusion matrices, feature importance, and model comparison are built into the dashboard itself.

06

Synthetic scenario engine

2,000 customers over 180 days at 15-minute resolution with regional presets and multiple theft scenarios.
