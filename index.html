FAQ

What changed on `main`?

Where did the desktop version go?

It was intentionally removed from `main` and preserved in the `desktop-local` branch.

Why are there no old screenshots anymore?

This branch should not present desktop visuals as if they were the future web product. The landing now waits for approved web assets instead.

How do I launch the current web app?

Run `START_MASS_AI.bat` or `START_MASS_AI_WEB.bat`, both of which now open the web runtime.
