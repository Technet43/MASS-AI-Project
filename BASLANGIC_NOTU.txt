MASS-AI Web Branch Notu
=======================

Bu branch artik web odakli ana hatti temsil ediyor.

Hizli baslangic:
1. `INSTALL_REQUIREMENTS.bat` calistir.
2. `START_MASS_AI.bat` veya `START_MASS_AI_WEB.bat` ile web uygulamasini ac.
3. `RUN_SMOKE_TESTS.bat` ile temel kontrol yap.

Yapi:
- `project\web` aktif web uygulamasi
- `project\core` ortak is mantigi
- `desktop-local` eski yerel/desktop surumu

Not:
Eski desktop ekran goruntuleri ve desktop calistirma dosyalari `main` branch'inden cikarildi.
