# MASS-AI

This repository is now organized around two clear app versions:

- `old_desktop/`
  The first desktop version built with Tkinter
- `new_web/`
  The newer Streamlit and web-facing version
- `shared/`
  Common engine, data, and tests used by both versions

## Quick Use

- Launcher:
  `START_MASS_AI.bat`
- Old desktop:
  `old_desktop/START_MASS_AI_DESKTOP.bat`
- New web:
  `new_web/START_MASS_AI_WEB.bat`
- Install dependencies:
  `INSTALL_REQUIREMENTS.bat`
- Smoke tests:
  `RUN_SMOKE_TESTS.bat`

## Notes

- Website files are under `new_web/site/`
- Shared datasets are under `shared/data/`
- Archived project notes and supporting documents are under `docs/`
